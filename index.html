Introductie Vakken Rooster Blokken Versnellen

Welkom!

Hier kan je allemaal informatie vinden over het eerste jaar op Curio!

Wat ga ik hier leren?

Je leert hier van alles. Als eerste begin je met simpele blokjes code bij elkaar te slepen, maar al snel begin je echt code te schrijven. Binnen een half jaar kan je al best veel! Je gaat in het eerste jaar vooral standaard web-development leren en een beetje Python/C#.

Welke vakken krijg ik allemaal?

WEB

WEB is precies wat je denkt dat het is. In deze lessen leer je vooral web-development. Je leert HTML code, styling, en ook back-end.

NATIVE

Hier leer je vooral echt software programmeren. Eerst start je met simpele Python scripts, en later leer je ook echt hele programma's maken!

PRA

Hier leer je echt vooral in groepjes werken, en ga je met kleine groepjes grote applicaties/websites maken. Teamwork is heel belangrijk hier.

PRO

Hier leer je eigenlijk alles dat te maken heeft met programmeren.. zonder het programmeren. Je leert dus in groepjes werken, ook leer je hoe je goed moet plannen en designen.

DIV

DIV staat voor Digitale Vaardigheden. Hier leer je vooral meer met programma's zoals Excel en Word omgaan, en leer je veiliger te zijn online.

Generiek

Programmeren is natuurlijk heel leuk, maar je hebt ook nog wat "saaiere" vakken zoals Engels, Nederlands en Rekenen. Hier krijg je vooral herhaling van wat je al hebt geleerd op school.
